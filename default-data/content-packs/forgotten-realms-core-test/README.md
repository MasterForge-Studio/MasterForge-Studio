# Forgotten Realms Core Test Pack

Internal MasterForge Studio v0.2 alpha test DLC.

## Purpose

This pack is designed to test:

- World → Region → Location hierarchy
- Theme inheritance
- Background/scene inheritance
- Location-specific backgrounds
- Region fallback backgrounds
- World fallback backgrounds
- Basic encounters, loot, weather, vendors, NPCs and factions

## Important

This pack is for local/internal testing only. Do not sell or publicly distribute this pack using protected third-party setting names.

## Image Structure

Scene placeholder images are included so UI switching can be tested immediately.
Replace them later with generated artwork using IMAGE_PROMPTS.md.

## Suggested Resolver Order

Location background → Region background → World background → App default.

Location theme → Region theme → World theme → Manual fallback.
