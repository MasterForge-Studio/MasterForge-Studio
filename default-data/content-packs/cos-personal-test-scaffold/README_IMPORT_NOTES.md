# Curse of Strahd Personal Test Scaffold

Private-use MasterForge testing pack only. Not for distribution.

This pack intentionally contains original short placeholders, relationship scaffolding, names, and organisational structure only. It does not include official adventure prose, boxed text, stat blocks, maps, encounter text, or artwork.

Use it to test MasterForge v0.2 entity/relationship/asset systems, then verify and fill your personal notes from the physical book you own.

Current v0.1 importer will import worlds/locations/regions/npcs/factions/weather/encounters/loot.
The entities/relationships folders are included for v0.2 testing once the importer supports them.
